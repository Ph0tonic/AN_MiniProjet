Aim of the project

This project's main aim is to implement a basic ant simulator and watch complex global system made of a lot of ant. The behaviour of those ants is pretty simple but all together they are stronger.

Execution

The execution has been made through multiple steps :

  • Draw map
  • Random move for ant
  • Going back home with knowing of the terrain
  • Following pheromones to find out some food sources
  • Going back home knowing the direction without taking in account their knowledges of the terrain
  • Modification of pheromone system to use the gradient
  • Improve ant to see at a certain distance
